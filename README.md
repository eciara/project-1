# Projekt
Projekt strony internetowej zbudowany w oparciu o plik graficzny

Po rozpakowaniu instalujemy zależności poleceniem

```bash
npm i
```

```bash
npm run build 
```
Tworzy nowy build produkcyjny, usuwając stare pliki

```bash
npm run start
```
